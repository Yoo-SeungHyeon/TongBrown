0. 사이트의 성질을 10개씩 정의한다. (1~3번 키워드 = 메인 키워드, 4~10번 키워드 = 서브 키워드)
================================
| A 1.--- 2.--- 3.--- ... 10.--- |
| B 1.=== 2.=== 3.=== ... 10.=== |
| .					 |
| .					 |
| .					 |
================================

1. 원하는 키워드를 입력한다. ex)야구
2. 원하는 키워드와 동일한 성질을 가진 사이트를 나열한다. = 야구라는 키워드가 포함된 사이트를 모두 찾는다.
3. 평균 조회수가 많은 사이트 + 메인 키워드, 서브 키워드를 반영해서 순서로 보여준다. 
4. 원하는 사이트를 선택하면 그 사이트의 추천 키워드를 보여주고, 원하는 문구(키워드)를 입력하면 예상 조회수를 알려줌.
