1. 크롤링 대상 : 네이버 상위 100개의 카페
2. 크롤링 세부사항 : 전체 게시글 -> 명사만 
3. 카페별 속성 지정 규칙
	3-1. 언급량(언급된 횟수 = 단어가 포함된 게시글 수) 상위 10% (상위 10%의 근거는 언급량의 평균과 표준편차를 활용한 정규화)
	3-2. 3-1을 만족하는 명사 중 '평균' 조회수가 높은 상위 10개의 글
	+최근에 언급이 많이 됐으면 그에 따른 가중치를 주는 것도 괜찮은듯. -> 애초에 추출을 최근 1년 또는 한달만 하는 것도 하나의 방법. + 날짜에 상관없이 추출 페이지를 지정하는 것도 좋을듯.(네이버 카페 기준 한 페이지에 15개의 게시물)

4. 테이블 속성
	4-1. 추출 테이블 [카페명(url_code랑 같은 역할), 제목, 조회수, 작성일자]
	4-2. 명사 테이블 [카페명, 명사, 조회수, 작성일자]
		4-2-1. 언급량 확인 방법  : 'SELECT 명사,count(*) FROM 명사_테이블 GROUP BY 명사' 이런 모양이면 
						 명사별 언급량 확인 가능할 듯
		4-2-1. 평균 조회수 확인 방법 : 각 명사별로 '총 조회수 / 언급량' 하면 평균 조회수를 구할 수 있을 듯.
	4-3. 메인 테이블 [카페명, 키워드1, 키워드2, ... ,키워드10]
